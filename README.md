# Transfinity Agent

An optional JVM Java agent that enhances gameplay behavior for the Transfinity mod (DOOM & Titans inspired).

---

## ⚠️ Important

This is **NOT required** to use the mod.
It is an optional enhancement that runs at JVM startup.

---

## 📥 Installation

1. Download the latest `transfinity-agent.jar` from the Releases section
2. Open your modpack in CurseForge
3. Click the **3 dots → Profile Options**
4. Add this to **Additional Arguments**:

```
-javaagent:"C:\Users\YourName\Downloads\transfinity-agent.jar"
```

---

## 🧠 What it does

* Enhances gameplay mechanics
* Applies runtime behavior improvements
* Works alongside Forge / NeoForge

---

## ❗ Notes

* Tested on Minecraft 1.20.4+
* May not work on older versions
* Safe to remove at any time
